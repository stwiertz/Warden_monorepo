## Challenges

- **Google Fonts 403** in build environment forced mid-story pivot from `next/font/google` to `next/font/local` with `@fontsource-variable/inter` woff2 files
- **Code review consistently caught quality issues** — fake test assertion (high severity), redundant CSS, breakpoint inconsistencies, missing aria-describedby
- **Planning artifacts config mismatch** — `config.yaml` pointed to `_bmad-output/planning-artifacts/` but files were at `_bmad/planning-artifacts/`. Discovered and resolved during this retrospective.
- **Firebase `auth/invalid-api-key` error** carried forward as unresolved debt (since resolved by Steph)

## Key Insights

1. **Code reviews catch real issues** — significant problems were found in every story's review. Code review must remain mandatory.
2. **Design system investment in early stories pays dividends later** — upfront CSS variable work in Story 1-2 made Story 1-3 straightforward.
3. **Config/file alignment matters** — verify BMAD config paths match actual file locations after any install/upgrade.
4. **Carrying technical debt forward is risky** — the Firebase API key issue from Epic 1 would have been a blocker for Epic 2 if not resolved.
